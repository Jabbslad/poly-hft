# agents.md

Workflow rules for AI agents working on this codebase.

## Commit Policy

- Commit and push to GitHub after each todo list item is completed
- Use descriptive commit messages that reference the completed task
- Ensure all pre-commit hooks pass before pushing

## Todo List Workflow

1. Mark task as `in_progress` before starting work
2. Complete the task
3. Mark task as `completed`
4. Commit changes with descriptive message
5. Push to GitHub
6. Move to next task
